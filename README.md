# Hi

Personal site built using (hugo)[https://gohugo.io/], using (codex)[https://github.com/jakewies/hugo-theme-codex] theme.

## Quickstart

Install Hugo:

```bash
brew install hugo
```

Adding a new page:

```bash
hugo new content posts/my-first-post.md
```

Run development server:

```bash
hugo server -D
```

Deployment:

```bash
./deploy.sh
```
